# **You Can use following commands:**
# Adding a new task
add "Buy groceries"

# Updating tasks
update 1 "Buy groceries and cook dinner"

# Deleting tasks
delete 1

# Marking a task as in progress or done
mark-in-progress 1

mark-done 1

# Listing all tasks
list

# Listing tasks by status
list done

list todo

list in-progress


# To get the list of commands on the go
help
